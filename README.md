# SQL Learning Platform Project

This project demonstrates fundamental and advanced SQL concepts using a custom-designed database schema inspired by an online learning platform.

## 📌 Overview

The project includes:

* Database schema definition
* Sample data for testing
* A collection of SQL queries ranging from basic to advanced level

The tasks are inspired by university coursework but fully redesigned with original scenarios and data.

---

## 🗂️ Project Structure

```
sql-learning-platform/
├── schema.sql      -- Database structure
├── insert.sql      -- Sample data
├── sheet6.sql      -- Basic queries
├── sheet7.sql      -- Advanced queries
├── sheet8.sql      -- Complex queries & analysis
└── README.md
```

---

## ⚙️ How to Run

1. Create the database schema:

```
SOURCE schema.sql;
```

2. Insert sample data:

```
SOURCE insert.sql;
```

3. Run query files:

```
SOURCE sheet6.sql;
SOURCE sheet7.sql;
SOURCE sheet8.sql;
```

* Filtering and sorting

* Recursive queries (CTE)
* Views
* Query optimization concepts
